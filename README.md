# Brain Tumor Classification

## Overview

A Deep Learning model to classify a brain tumor based on MRI images. A brain tumor is an abnormal mass of cells that grows in or around the brain.. The goal is to train a high accuracy model without underfitting or overfitting. 

## Dataset

This project uses the dataset from Masoud Nickparvar's Brain Tumor MRI Dataset (https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset). It contains 4 classes with 5600 files for the training set and 1600 files for the testing set.

## Model

Convolutional Neural Network
